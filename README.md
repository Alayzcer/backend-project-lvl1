## install

For playing you need to have a nodejs version 15. How to install nodejs see on https://nodejs.org
for install games use make:

- make install

Or you can use Docker:

- make run

Games are available:

- brain-games
- brain-even
- brain-calc
- brain-gcd
- brain-progression
- brain-prime
